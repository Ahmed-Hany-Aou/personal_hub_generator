SDK Integration Demo

This page demonstrates how a 3rd-party SaaS app can use the Elite Hub SDK to embed our designer engine.

const sdk = new EliteHub('elite-dev-123');
sdk.embed(document.getElementById('hub-preview'), {
    fullName: "SDK User",
    activePageSlug: "index",
    pages: [{ 
        slug: "index", 
        name: "Home", 
        theme: "cyber",
        components: [...] 
    }]
});
